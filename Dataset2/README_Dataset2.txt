1. Supplementary Table S2
Species list of dataset 2 used for GeneMiner performance and comparison.

2. shallow_ref
This folder stores common phylogenetic markers (ITS, matK, psbA, and rbcL) from five species (Bupleurum chinense, Cnidium monnieri, Cyclospermum leptophyllum, Ligusticum jeholense, and Pimpinella diversifolia) for GeneMiner's reference sequences. For details, see Supplementary Table S2.


3. shallow_gold
This folder stores common phylogenetic markers (ITS, matK, psbA, and rbcL) from five species (Bupleurum chinense, Cnidium monnieri, Cyclospermum leptophyllum, Ligusticum jeholense, and Pimpinella diversifolia), used as the gold standard. For details, see Supplementary Table S2

4. ref_hybpiper.fasta
The reference sequences are used for HybPiper.

5. config_for_novoplasty.txt 
parameter settings file for NOVOPlasty.


6. species.list
Species name list

7. gene.list
Common phylogenetic markers list

8. work.sh
The file records the commands to run the Dataset2.

Note1: You should install GeneMiner, HybPiper 2.01, NOVOPlasty4.3.12
Note2: Before running, please pay attention to the parameters selection and path setting.
